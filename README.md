# task02_Anshima

Data Classification Using AI
Project Overview

This project demonstrates the implementation of a Machine Learning classification model using Python and the Iris Dataset. The objective is to train a model that can classify flowers into different species based on their physical characteristics.

The project covers the complete supervised learning workflow, including data loading, data preprocessing, model training, prediction, evaluation, and visualization.



Objectives:
1. Load and understand a dataset
2. Split data into training and testing sets
3. Train a classification model
4. Evaluate model performance
5. Visualize results using graphs
6. Predict flower species based on user input
7. Technologies Used
8. Python
9. Scikit-Learn
10. NumPy
11. Pandas
12. Matplotlib
13. Seaborn



Dataset

The project uses the Iris Dataset, a well-known dataset in Machine Learning.

Features:

1. Sepal Length
2. Sepal Width
3. Petal Length
4. Petal Width



Target Classes:

1. Setosa
2. Versicolor
3. Virginica



Machine Learning Algorithm: 

Decision Tree Classifier

A Decision Tree Classifier is used to train the model and classify flower species based on feature values.


Project Workflow
1. Load the Iris Dataset
2. Split the dataset into training and testing sets
3. Train the Decision Tree model
4. Make predictions on test data
5. Calculate model accuracy
6. Generate a classification report
7. Display graphs for analysis
8. Predict flower species from user input



Visualizations :

The project generates the following graphs:

1. Confusion Matrix
Displays the number of correct and incorrect predictions made by the model.

2. Feature Importance Graph
Shows the importance of each feature used by the Decision Tree Classifier.

3. Actual vs Predicted Graph
Compares actual class values with model predictions.
